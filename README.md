# Next.js project boilerplate

[![CircleCI](https://circleci.com/gh/willnguyen1312/nam-next.svg?style=svg)](https://circleci.com/gh/willnguyen1312/nam-next)

## Setup

- git clone https://github.com/willnguyen1312/nam-next.git {folder_name}
- cd {folder_name}
- yarn install

## Some routes under construction for demo purposes

1. home page and about page -> react-intl
2. counter page and other page -> redux

## List of commands

1. Develop: yarn dev
2. Build: yanr build
3. Test: yarn test
4. Run production build: yarn build && yarn start
5. Docz: yarn docz:dev
